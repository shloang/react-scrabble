# Russian Scrabble (Эрудит) Game

## Overview

This is a multiplayer Russian Scrabble (Эрудит) game built as a full-stack web application. The game supports up to 3 players who compete by forming words using Cyrillic letter tiles on a 15×15 game board. Players take turns placing tiles, with special squares providing scoring multipliers (triple word, double word, triple letter, double letter). The application features real-time game state synchronization, turn-based gameplay with timers, and word validation, simple login-continue, word search, highlights, and move history.
